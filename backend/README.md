# Ninvax Backend

This directory provides a basic Express server skeleton intended for Supabase and Stripe integration.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```

Environment variables used:

- `PORT`
- `SUPABASE_URL`
- `SUPABASE_KEY`
- `STRIPE_SECRET`
